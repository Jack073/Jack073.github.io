Amigo is a discord bot made using discord.js for server Thy Amigos. To host a version of this bot yourself follow the instructions below:

  1. Install the latest version of node.js
  2. Make sure all dependencies are installed by running the command npm install
  3. Change config.js.example to config.js and insert the bot token and the Bot Owner's ID
  4. Click run.bat and you should be good to go.

Instead, if you would like to use a hosted version of the bot, we have a hosted version of this bot which you can invite to your server here
